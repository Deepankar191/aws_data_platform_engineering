# =============================================================================
# Credit Decision Data Platform — project-level Python requirements
# =============================================================================
# The complete dependency set for the whole repo, pinned to the runtime targets
# (Glue 4.0 / Spark 3.3, MWAA Airflow 2.8, Python 3.10–3.11). Per-component
# requirements.txt files still exist for packaging a single unit (a Lambda zip, a
# Glue bundle); this file is the union, for a full local dev / CI environment:
#
#     python3 -m venv .venv && source .venv/bin/activate
#     pip install -r requirements.txt
#
# System (non-pip) deps: Graphviz `dot` (for `diagrams`); optional `pandoc` (PDFs).
#
# NOTE — Apache Airflow. Airflow must be installed with its official constraints
# file or pip may resolve incompatible transitive versions. If you hit a resolver
# conflict, install Airflow first, in its own step, then the rest:
#   pip install "apache-airflow==2.8.4" \
#     --constraint "https://raw.githubusercontent.com/apache/airflow/constraints-2.8.4/constraints-3.10.txt"
# Airflow (MWAA) and Spark (Glue) are provided by the managed runtime in AWS — you
# only need them locally to run/parse those layers.
# =============================================================================
